## ğŸš€ Mission Start (Installation)

To deploy this unit locally:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/ArashadDodhiya/anime-portfolio.git
    cd anime-portfolio
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

    ```bash
    npm run dev
    ```

4.  **Access the Interface**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.
